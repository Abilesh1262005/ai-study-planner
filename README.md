# 📚 AI Study Planner

A smart study planning web app that helps students organize their exam preparation with AI-based daily hour suggestions.

## 🚀 Features

- ✅ AI-based study hour calculation  
- ✅ Task completion tracking  
- ✅ Progress doughnut chart  
- ✅ GitHub-style activity heatmap  
- ✅ Dark / Light mode  
- ✅ LocalStorage persistence  
- ✅ Responsive premium UI  

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- Chart.js
